# Login Issues Troubleshooting

## Common Login Problems

### 1. Incorrect Credentials
- Verify email and password are correct.
- Passwords are case-sensitive.
- Use "Forgot Password" to reset if needed.

### 2. Account Lockout
- Accounts lock after 5 consecutive failed login attempts.
- Lockout duration: 30 minutes.
- Use "Forgot Password" to reset — this also unlocks the account.

### 3. Browser Issues
- Clear browser cache and cookies.
- Disable browser extensions that may interfere (ad blockers, privacy tools).
- Try incognito/private browsing mode.
- Use an up-to-date browser: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+.

### 4. Session Expired
- Sessions expire after 24 hours of inactivity.
- Re-login to create a new session.

### 5. Single Sign-On (SSO) Issues
- Verify SSO configuration with your organization's IT team.
- Ensure your email domain matches the SSO configuration.
- Contact support for SSO-specific troubleshooting.

## API Login
For API authentication, use OAuth 2.0 client credentials flow. See the API Authentication guide for details.
